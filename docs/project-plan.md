# Dishwasher Safe? Micro-site

## Goal

Spin up a second SEO micro-site using the Laundry Settings pattern, but focused on one household question: “Can this go in the dishwasher?”

## Current scope

- Astro static site
- Vercel deployment
- structured item/material data
- item pages
- material pages
- question-shaped can-you pages
- sitemap, robots.txt, canonical URLs
- FAQ schema on item pages
- Google Analytics / Search Console env hooks

## Not in scope yet

- Domain purchase or connection
- Search Console verification/submission
- Analytics property setup
- Ads or affiliate links

## MVP keyword patterns

- can you put {item} in the dishwasher
- is {item} dishwasher safe
- are {material} dishwasher safe
- dishwasher safe top rack only {item}
- can {brand/product} go in dishwasher

## Expansion candidates

- More specific bottle brands and lids
- Kids and baby feeding gear
- Cookware by material
- Appliance accessories: air fryer, blender, Instant Pot, food processor
- “Top rack or bottom rack?” pages
- “Can I use heated dry?” pages
